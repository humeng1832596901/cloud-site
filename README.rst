易云（cloud-site）
功能简介
-------
- 权限划分；
    - 异步多文件上传；
    - 群组功能，邀请码加入群组；
    - 文件分享；
    - 音频、纯文本、图片在线直接查看，专门优化代码查看效果；
    - 消息反馈；
安装方式：
--------
  1.uwsgi + nginx
  2.gunicorn + nginx
其他：
----
  1.默认注册账户是普通用户，第一次使用，创建完成后，想要获得超级权限需要将对应的字段手动修改
  2.数据库默认使用SQLit，在实际中建议使用MySQL
